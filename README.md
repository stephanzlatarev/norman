Nidamasin Norman

### Start on Windows

Start StarCraft II API

```
cd "C:\games\StarCraft II"
start /D .\Support .\Versions\Base88500\SC2.exe -listen 127.0.0.1 -port 5000
```

Start Norman

```
npm start
```

### Run in SC2 AI Arena

* Zip to `norman.zip`
* Upload at https://aiarena.net/bots/518/edit
